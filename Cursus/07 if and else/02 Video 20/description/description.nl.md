## ALS/DAN
<b>ALS</b> <i>booleaanse expressie</i> <b>DAN</b><br>
   taak 1 <br>
   taak 2 <br>
   taak 3 <br>
De booleaanse expressie is een voorwaarde vandaar dat het <code>if</code>-statement ook conditional statement genoemd wordt.
